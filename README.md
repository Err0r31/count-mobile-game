# Count Mobile Game

**Count Mobile Game** — это мобильная игра на реакцию, внимательность и скорость счёта. Игроку необходимо как можно быстрее и точнее решать математические примеры, зарабатывая очки и продвигаясь по уровням сложности.

Игра ориентирована на тренировку:
- концентрации внимания,
- скорости реакции,
- базовых арифметических навыков.

---

## Основные возможности

- Решение математических примеров на время  
- Подсчёт очков за правильные ответы  
- Система уровней сложности  
- Переход между экранами через Expo Router  
- Хранение прогресса игрока  
- Адаптация интерфейса под мобильные устройства  

---

## Как устроен проект

- Используется **file-based routing** (`app/`) через Expo Router  
- Каждый экран — отдельный маршрут  
- Игровая логика разделена на:
  - UI
  - состояние
  - генерацию заданий
- Код полностью написан на **TypeScript**
- Используется конфигурация для сборок через **EAS**

---

## Стек технологий

- ESLint

---

## Структура репозитория

- `app/` — экраны и маршруты приложения  
- `components/` — переиспользуемые UI-компоненты  
- `assets/` — изображения, иконки, шрифты  
- `storage.ts` — работа с состоянием и хранением данных  
- `ui.ts` — UI-константы и вспомогательные функции  
- `app.json` — конфигурация Expo  
- `eas.json` — настройки сборки EAS  
- `jest.config.js`, `jest.setup.js` — конфигурация тестов  
- `eslint.config.js` — правила линтинга  
- `tsconfig.json` — конфигурация TypeScript
